# LumosCore — Frontend Showcase

A design showcase of LumosCore's upcoming web interface, in both **desktop** and **mobile** versions.

## Live preview

**https://exquisite-kataifi-cd435d.netlify.app**

- Opens the **desktop** interface on computers and the **mobile** interface on phones automatically.
- To view a specific version on any device, add `/?desktop` or `/?mobile` to the link.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Entry page — detects the device and routes to the matching version |
| `desktop.html` | Desktop interface showcase |
| `mobile.html` | Mobile interface showcase |

Each showcase is a single self-contained HTML file (styles and assets inlined); the only external dependency is Google Fonts.

## About LumosCore

LumosCore is a cross-chain DEX / AMM / launchpad built on **Stellar** and **XRPL**. Its **Liquidity Routing Protocol (LRP)** settles cross-chain swaps by routing through liquidity rather than wrapping or minting assets.

- Website: https://lumoscore.com

## Status

This repository is a **preview of the upcoming interface** — a design showcase, not the production application.
